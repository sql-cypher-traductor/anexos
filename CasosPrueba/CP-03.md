| **CAMPO**                    | **DESCRIPCIÓN**                                              |
| ---------------------------- | ------------------------------------------------------------ |
| **ID**                       | CP-03                                                        |
| **Historia de Usuario**      | QTE-04                                                       |
| **Nombre**                   | Traducir consultas INSERT                                    |
| **Cumple (Sí/No)**           | Sí                                                           |
| **Descripción de la Prueba** | Verificar que el sistema permita traducir consultas `INSERT` correctamente. |
| **Precondiciones**           | 1. El usuario debe estar autenticado en el sistema<br>2. El usuario debe tener una conexión a Neo4j almacenada y activa. |
| **Pasos**                    | 1. Acceder a la pantalla de traducción de consultas<br>2. Seleccionar una conexión a Neo4j activa<br>3. Ingresar una sentencia SQL `INSERT` válida<br>4. Dar clic en el botón `Traducir` |
| **Datos de Prueba**          | Sentencia: `INSERT INTO Usuario (nombre, edad) VALUES ('Usuario Nuevo', 20);` |
| **Resultados Esperados**     | El sistema indica el equivalente en lenguaje Cypher junto a la sentencias SQL ingresada. |
| **Resultados Obtenidos**     | El sistema mostró un mensaje de éxito de consulta traducida y la consulta equivalente en lenguaje Cypher junto a la consulta SQL ingresada, además del tiempo de traducción. |
