| **CAMPO**                    | **DESCRIPCIÓN**                         |
| ---------------------------- | --------------------------------------- |
| **ID**                       | CP-06                                   |
| **Historia de Usuario**      | QTE-06                                  |
| **Nombre**                   | Registrar consulta en el historial      |
| **Cumple (Sí/No)**           | Sí                                      |
| **Descripción de la Prueba** | Validar almacenamiento automático de las consultas en el historial. |
| **Precondiciones**           | 1. El usuario debe estar autenticado en el sistema<br>2. El usuario debe tener una conexión a Neo4j almacenada y activa. |
| **Pasos**                    | 1. Seleccionar una conexión a Neo4j activa<br>2. Ingresar una sentencia válida<br>3. Dar clic en el botón `Traducir y Ejecutar`<br>4. Dirigirse al Historial de Consultas. |
| **Datos de Prueba**          | Sentencia: `SELECT * FROM Employee WHERE title = 'Sales Representative';` |
| **Resultados Esperados**     | El sistema indica la consulta registrada en el historial con etiqueta `EJECUTADA`. |
| **Resultados Obtenidos**     | El sistema indica la consulta ejecutada en el historial con las estadísticas de ejecución y la etiqueta `EJECUTADA`. |
