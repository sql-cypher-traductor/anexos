            <b>HISTORIA DE USUARIO: </b><br>
            <b>Como</b> usuario,<br>
            <b>Quiero</b> traducir sentencias SELECT, CREATE, UPDATE o DELETE simples a lenguaje Cypher,<br>
            <b>Para</b> interactuar con bases de datos orientadas a grafos sin tener que aprender un nuevo lenguaje de consulta.
        </td>
    </tr>
    <tr>
        <td colspan=2>
            <b>CRITERIOS DE ACEPTACIÓN: </b><br>
            <ul>
                <li>El usuario debe agregar una sentencia SQL válida para ser traducida.</li>
                <li>Las sentencias soporta cláusulas, operadores y funciones básicas.</li>
                <li>Al ingresar una sentencia válida, el sistema devolverá su equivalente en lenguaje Cypher.</li>
                <li>Las consultas traducidas serán agregadas al historial de consultas.</li>
            </ul>
        </td>
    </tr>
    <tr>
        <td colspan=2>
            <b>OBSERVACIONES: </b><br>
            <ul>
                <li>Verificar que la consulta a ser traducida no contenga errores de sintaxis.</li>
                <li>La traducción a Cypher aparecerá junto a la sentencia SQL correspondiente.</li>
            </ul>
        </td>
